PID Control Equation

For this lab, we will be using the PID control equation to control the position of our robot. We use the equation from lecture shown below:

The PID equation uses 3 main constants: Kp (proportional constant), KD (derivative constant), and Ki (integral constant). It uses these constants and the error to calculate the new PID input value. The Proportional control adjusts output based on error magnitude, with larger errors causing stronger responses. Integral control accumulates error over time, maintaining correction even when the error momentarily disappears, ensuring steady-state accuracy. Derivative control responds to error rate changes, essentially predicting future behaviorâ€”like slowing a robot approaching a wall too quickly.
