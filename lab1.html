Lab 1: Artemis and Bluetooth Set-up

Lab Objective

The purpose of Lab 1, was to get familiar with the Artemis Nano board and set-up the ble-controller to allow communicating with the board via blutooth.

Prelab Set-up

To set-up the lab I first downloaded the Arduino IDE and connected to the Artemis Nano board. After that, I ran some of the example code to test out uploading and running code on the board. I also got familiar with the Serial Monitor and baud rates.

Additional 5000-level Tasks

The additional task was to blink the LED on the artmemis whenever a Middle-C note was played. To achieve this code, I combined the sample blink and PDM codes and modified it such that the LED would blink if a frequency equal to that of Middle-C is heard. I allowed for small variations by setting the range of detection to 260 to 265 Hz. (Middle-C is generally considered to be 262 Hz). As can be seen in the video, the LED only blinks when a Middle-C is played and not any other note.
